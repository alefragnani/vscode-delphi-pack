FAQ

Quick answers for common questions.

Can I install the Pack if I already have any of the included extensions?

Yes. You can install the extension pack even if you already have some of the included extensions. VS Code will handle any overlaps appropriately.

Can I expand the extension pack with my own extensions?

No. The extension pack cannot be expanded with additional extensions.

How can I support this project?

You can support development through PayPal, GitHub Sponsors, or Patreon.

Does it work on VS Code forks like Cursor, Windsurf, and Antigravity?

Yes. As a standard extension pack, it can be installed on compatible VS Code-based editors that support Marketplace or Open VSX extensions.
